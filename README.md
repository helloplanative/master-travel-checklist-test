# Master Checklist für Reisen – Version 2

Diese Version enthält eine statische Website plus Netlify Function.

Die App befüllt verfügbare Basisdaten und fixe Länderfakten. Sensible Punkte wie Visum, Pass, Gesundheit, Zoll und Führerschein werden nicht erfunden, sondern mit Quellenhinweis und Vertrauensgrad als offiziell zu prüfen markiert.

Für echte produktive Nutzung braucht man später eine professionelle Datenquelle wie Timatic oder eine andere Travel-Requirements-API.
